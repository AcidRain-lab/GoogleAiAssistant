Опис бібліотеки WebLoginBLL
Призначення бібліотеки
Ця бібліотека призначена для управління користувачами, їхніми ролями та авторизацією в застосунку. Вона надає методи для створення, оновлення, видалення та перевірки користувачів, а також для виконання операцій автентифікації та авторизації.

Склад класів
1. DTO (Data Transfer Objects)
UserDTO

Представляє дані користувача, які передаються між шарами застосунку.
Властивості:
Id: Унікальний ідентифікатор користувача.
UserName: Ім'я користувача.
Email: Електронна пошта користувача.
RoleName: Назва ролі, що належить користувачу.
IsActive: Статус активності користувача.
RoleDTO

Описує дані про роль користувача.
Властивості:
Id: Унікальний ідентифікатор ролі.
Name: Назва ролі.
LoginDTO

Використовується для передачі даних входу користувача.
Властивості:
UserName: Ім'я користувача.
Password: Пароль.
2. Сервіси
AuthService

Відповідає за операції автентифікації.
Основні методи:
SetCookieAuthentication: Налаштовує автентифікацію через кукі.
Logout: Виконує вихід користувача.
VerifyPassword: Перевіряє правильність пароля.
UserService

Відповідає за управління користувачами.
Основні методи:
GetAllUsersWithRoles: Отримує всіх користувачів разом із їхніми ролями.
CreateUser: Створює нового користувача.
UpdateUser: Оновлює дані користувача.
DeleteUser: Видаляє користувача.
RoleService

Управляє ролями користувачів.
Основні методи:
GetAllRoles: Отримує всі ролі.
CreateRole: Додає нову роль.
DeleteRole: Видаляє роль.
3. Хелпери
AuthHelper

Налаштовує автентифікацію та авторизацію.
Методи:
GetCombinedHeaders: Генерує заголовки для автентифікації.
JwtAuthHelper

Працює з JWT-токенами.
Методи:
GenerateJwtToken: Генерує токен для користувача.
CookieAuthHelper

Налаштовує автентифікацію через кукі.
4. Конфігурація
JwtSettings

Зберігає параметри конфігурації для роботи з JWT:
SecretKey: Секретний ключ для підпису токенів.
Issuer: Видавець токенів.
Audience: Цільова аудиторія токенів.
AuthenticationSettings

Конфігурація автентифікації:
EnableJwt: Вмикає/вимикає JWT-автентифікацію.
EnableCookies: Вмикає/вимикає кукі-автентифікацію.
5. Профілі мапінгу
MappingProfile
Налаштовує мапінг між сутностями та DTO за допомогою AutoMapper.
6. Інструменти безпеки
SecurePasswordHasher
Хешує паролі та перевіряє їхню коректність.
