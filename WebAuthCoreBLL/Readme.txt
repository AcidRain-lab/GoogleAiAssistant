README: Бібліотека для автентифікації та авторизації
Ця бібліотека забезпечує комплексне рішення для автентифікації та авторизації в ASP.NET Core застосунках. Вона інтегрує підтримку JWT (JSON Web Token) та автентифікації через cookies, що дозволяє розробникам створювати безпечні застосунки з ролевим контролем доступу.

Призначення
Бібліотека призначена для:

Управління автентифікацією користувачів через cookies або JWT токени.
Реалізації ролевого контролю доступу до дій та інтерфейсів.
Безпечної обробки паролів та перевірки їх хешованих значень.
Основні класи
1. AuthHelper
Відповідає за налаштування автентифікації та авторизації.
Підтримує cookies та JWT.
Визначає політики доступу.

2. CookieAuthHelper
Налаштовує cookies для автентифікації.
Обробляє події перенаправлення у разі неавторизованих або заборонених запитів.

3. JwtAuthHelper
Керує створенням і перевіркою JWT токенів.
Використовується для API авторизації.

4. SecurePasswordHasher
Забезпечує безпечне хешування паролів.
Використовується для створення та перевірки хешів паролів.

5. AuthenticationSettings
Конфігураційний клас для зберігання налаштувань автентифікації, таких як тип (JWT або cookies) та сервер.

6. JwtSettings
Містить параметри для генерації JWT токенів, включаючи секретний ключ, видавця та аудиторію.

7. AuthorizeRoleAttribute
Атрибут для перевірки ролей користувача.
Використовується для обмеження доступу до методів контролерів.

8. LayoutByRoleAttribute
Атрибут для динамічного налаштування layout'ів залежно від ролі користувача.

9. ApplicationConstants
Зберігає загальні константи, що використовуються в застосунку.

Встановлення та налаштування
Додайте залежності бібліотеки у ваш проєкт.
Налаштуйте AuthenticationSettings та JwtSettings у файлі конфігурації.
Використовуйте атрибути AuthorizeRoleAttribute та LayoutByRoleAttribute для контролю доступу та налаштування інтерфейсу.
